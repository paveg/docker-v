# docker-v
Simply try to use [v](https://vlang.io/) in Docker.

## The V Programming Language

**Simple, fast, safe, compiled language for developing maintainable software.**

https://vlang.io/

## Usage

### Build & Run

```bash
$ make build
$ make v # awesome v language!
```

also, use v REPL.

```bash
$ make run.v
```
